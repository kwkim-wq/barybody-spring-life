2. 왜 소모품으로 가는가

스프링을 반영구 부품으로 가려면 필요한 수명:

단계계산결과
세션당 왕복30회/분 × 10분300 회
1년300회 × 365일약 109,500 회
5년109,500회 × 5년약 547,500 회
안전계수 ×4547,500회 × 4약 2,190,000 회
필요 수명약 2,000,000 회 (200만 회)
  • 제조사 제시 10만 회는 위 필요치의 1/20 수준
  • 안전계수 없는 순수 필요치(55만 회)와 비교해도 1/5.5
  • → 반영구 부품으로는 성립하지 않으므로 소모품(교체품) 방식으로 전환

3. 소모품 방식에서 반드시 챙길 것

⚠️ 안전 (최우선)

소모품이라는 건 언젠가 반드시 파손된다는 뜻. 100kg 장력이 걸린 스프링이 갑자기 끊어지면 튕겨서 부상 위험.

  • 수명 도달 시 갑자기 파단되지 말고 장력이 서서히 떨어지는 특성 (파단 전 경고)
  • 파손되더라도 튕겨나오지 않게 구속/커버되는 구조

이 부분은 타협 불가.

교체 용이성

  • 고무밴드처럼 공구 없이 사용자가 직접 끼웠다 뺐다 가능해야 함
  • 프리로드 조절기구를 풀고 교체하는 동선이 간단한지 확인

교체품 공급 구조

  • 초기 구성에 여분 스프링 포함 (레퍼런스 슬라힙도 밴드 1세트 추가 증정)
  • 별매 교체품 가격·판매 채널 세팅
  • 소모품이니 단가를 낮춰 여러 개 넣는 게 유리

사전 안내 (CS 방어)

  • 상세페이지·매뉴얼에 "스프링은 소모품이며 권장 교체주기는 O년" 명시
  • 안 하면 "1년 만에 고장났다" 클레임이 그대로 들어옴

4. 제조사에 확인할 것

  1. 10만 회가 최고 저항(100kg) 기준인지 — 조건이 다르면 숫자 비교 자체가 무의미
  2. 수명 도달 시 파단인지, 장력 저하인지 — 안전 직결
  3. 교체용 스프링 단가와 최소 발주 수량
  4. (참고) 10만 / 50만 / 100만 회 구간별 단가 차이 — 소폭이면 상향 검토 여지
